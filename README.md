## Install
[Chrome](https://chrome.google.com/webstore/detail/gas-pod/diobnnjkbkdpdgenhlgjanifmaimmppp)

## Dev
```
yarn
yarn build
```
click `Load unpacked` button on `Chrome Extensions`, then select the `build` fold
